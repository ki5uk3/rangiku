# rangiku

simple bash script for recon.

<b>Pre Requirements</b>

1)Findomain<br>
2)Subfinder<br>
3)httprobe<br>
4)nmap<br>
5)knockpy<br>
6)Amass<br>
7)Assetfinder<br>
8)Httprobe<br>
9)Nuclei

<b>Set Permisson</b>

$ chmod +x rangiku.sh

<b>Use</b>

$ ./rangiku.sh [DOMAIN]

<b>Example</b>

$ ./rangiku.sh tiktok.com
